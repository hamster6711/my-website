Invading hamster java game

This is simple game was written in Java using Eclipse, it was developed in summer 2016. I worked on developing both the back end game algorithm, and front end framework to support the animation and graphical display.

Game rules: Try to scare away all the hamster falling from the sky and avoid the poop they dropped!

Illumination phoney mosaic

The app allows the crowd to connect their phones to a central server. It will then receive instructions on its role in the mosaic based on its location. Close up, it may just seem like it is showing a random colour; but far away, that phone acts as a pixel in the bigger picture. I worked with Android Studio to create the user end app. This was a hackathon project in 2015.
